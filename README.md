**Установка**

Сборка и запуск проекта выполняются по команде npm run start.

- `npm run dev` — запуска проекта в режиме разработки,
- `npm run start` — сборка и запуск проекта,
- `npm run build` — сборка стабильной версии.

Проект можно скачать себе и развернуть локально из гитхаб репозитория [messenger-gonec](https://github.com/Antonov2504/messenger-gonec), командой:

```html
$ git clone https://github.com/Antonov2504/messenger-gonec.git
```
